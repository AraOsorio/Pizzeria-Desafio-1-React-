# 🍕 Pizzería Mamma Mia

Proyecto desarrollado con React + Vite, para el módulo de React del bootcamp de Full Stack de Desafío Latam.

## 🚀 Descripción

Este proyecto consiste en una aplicación web para una pizzería, donde se muestran diferentes tipos de pizzas con sus ingredientes, precios y opciones de interacción.

## 🛠 Tecnologías utilizadas

- React
- Vite
- Bootstrap 5
- JavaScript
- CSS

## 🎨 Funcionalidades

- Navbar con opciones de navegación
- Header con imagen destacada
- Cards de pizzas dinámicas
- Formato de precios en CLP
- Diseño responsive
